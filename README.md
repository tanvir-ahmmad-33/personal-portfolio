# personal-portfolio

Personal portfolio with React + Tailwind CSS frontend and Laravel backend for dynamic content management.

## Overview

`personal-portfolio` is a full-stack portfolio web application built with **Laravel**, **Inertia.js**, **React JSX**, **Tailwind CSS**, and **Vite**.

The public side displays portfolio content such as profile details, skills, projects, experience, education, resume, and contact information. The admin dashboard allows the portfolio owner to manage content dynamically without editing the source code.

## Tech Stack

- Laravel
- Inertia.js
- React JSX
- Tailwind CSS
- Vite
- MySQL
- Laravel Authentication

## Features

### Public Website

- Home page
- About section
- Skills section
- Projects section
- Project details page
- Experience section
- Education section
- Contact form
- Resume download
- Social media links
